# my_vue


## 簡介
- npm 全名為 Node Package Manager，是 Node.js 的套件（package）管理工具

## 安裝
- 安裝 Node.js 會自動內建 npm，前往安裝 node.js https://nodejs.org/en
- 檢查是否安裝成功
    - node -v
    - npm -v

## 建立專案
- 初始化專案: npm init vue@latest
- npm install
- npm run dev

## 建立專案
- npm create vite@latest / npm init vite@latest
- npm install
- npm run dev
